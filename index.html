PROJECTS

3D Working Mouse Design

Used Solidworks form tools to design an original sleek and ergonomic computer mouse shell which could be outfitted with a circuit board from a broken computer mouse.

To the Moon

A game developed in the Godot game engine using python currently available on Google Playstore. To the Moon is an atmospheric game where the player uses touch gestures to jump up a series of randomly generated platforms in hopes of reaching the moon. All assets were 100% made personally, from the code to the sprites and composing an original version of clair de lune.

Canada Airport

Designed an airport program that read airplane flights from a textfile and displayed departure and arrival flights to Toronto.

Turntable

Used CAD knowlegde to design a turntable in Fusion 360 and printed it out using a 3D printer.
